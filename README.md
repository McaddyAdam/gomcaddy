# GoMcaddy — Local Food Ordering

GoMcaddy is a lightweight food ordering web app showcasing Nigerian restaurants. This workspace contains a React frontend and a FastAPI backend.

Maintained by Mcaddy Tech Solutions — contact: mcaddytechsolutions@gmail.com

Quick start (frontend):

```powershell
cd frontend
yarn install
yarn start
```

Quick start (backend):

```powershell
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --port 8000
```

For deployment, set environment variables in `backend/.env` and run production builds.
